# Componentes del Sistema de Emergencias y Seguridad Vial

Este documento detalla los componentes de cada página del sistema para crear un tutorial interactivo.

## 1. Panel de Seguridad Vial

**Descripción:** Panel principal para gestión de la seguridad vial, monitoreo de flotas y acceso a sistemas de control de tránsito.

**Componentes:**
- Tarjetas de servicios (ServiceCard)
  - Wisetrack: Sistema de Monitoreo ADAS
  - GPSChile: Sistema de Monitoreo de Flotas Navman
  - Gauss Control: Sistema de Control y Gestión de Fatiga en la Conducción
  - WebControl: Acreditación de personas y equipos
  - Explor-K: Sistema de gestión y control de tránsito para operaciones mineras
- Timeline de actualizaciones de organismos oficiales (TwitterTimeline)
- Sección de información de contacto
  - Supervisores de ruta
  - Centros de control
- Registros
  - Formulario de registro de incidentes (IncidentForm)
  - Formulario de control de camiones (CirculationForm)

## 2. Wisetrack

**Descripción:** Sistema de Monitoreo ADAS (Advanced Driver Assistance Systems).

**Componentes:**
- Interfaz de monitoreo de vehículos
- Mapas de seguimiento
- Informes y alertas
- Dashboard de indicadores de seguridad

## 3. GPSChile

**Descripción:** Sistema de Monitoreo de Flotas Navman.

**Componentes:**
- Interfaz de seguimiento de vehículos
- Mapas en tiempo real
- Alertas y notificaciones
- Reportes de rutas y recorridos

## 4. Gauss

**Descripción:** Sistema de Control y Gestión de Fatiga en la Conducción.

**Componentes:**
- Dashboard de alertas de fatiga
- Informes de descanso de conductores
- Estadísticas de patrones de conducción
- Configuración de alertas

## 5. WebControl

**Descripción:** Sistema de acreditación de personas y equipos.

**Componentes:**
- Registro de personal
- Gestión de credenciales
- Validación de documentación
- Control de accesos

## 6. Explor-K

**Descripción:** Sistema de gestión y control de tránsito para operaciones mineras.

**Componentes:**
- Mapa de tránsito en operaciones
- Gestión de rutas
- Control de accesos
- Informes de circulación

## 7. Repositorio (Document Panel)

**Descripción:** Gestión documental y repositorio de archivos del sistema.

**Componentes:**
- Buscador de documentos
- Categorías de documentos
- Visor de documentos
- Formulario de carga de documentos
- Historial de versiones

## 8. Registros (Records Panel)

**Descripción:** Sistema de registro de eventos, incidentes y reportes.

**Componentes:**
- Tabla de registros
- Filtros de búsqueda
- Formulario de nuevo registro
- Detalles de registros
- Exportación de datos

## 9. Simuladores (Simulators Panel)

**Descripción:** Plataforma de simulación para entrenamiento y pruebas.

**Componentes:**
- Selector de escenarios
- Controles de simulación
- Parámetros configurables
- Resultados y métricas
- Historial de simulaciones

## 10. Emergencias (Emergency Panel)

**Descripción:** Sistema de gestión de emergencias y respuesta a incidentes.

**Componentes:**
- Mapa de emergencias activas
- Formulario de reporte de emergencias
- Timeline de eventos
- Gestión de recursos
- Protocolos de actuación
- Sistema de notificaciones
- Dashboard de estado actual

## 11. Unidades (Emergency Units Panel)

**Descripción:** Gestión de unidades de emergencia y recursos disponibles.

**Componentes:**
- Listado de unidades
- Estado y ubicación de cada unidad
- Formulario de asignación
- Historial de despliegues
- Información de contacto
- Mantenimiento y disponibilidad

## 12. Personal

**Descripción:** Gestión del personal involucrado en emergencias y seguridad vial.

**Componentes:**
- Directorio de personal
- Gestión de turnos
- Cualificaciones y certificaciones
- Historial de asignaciones
- Contactos de emergencia
- Formulario de registro de personal

## 13. Dashboard

**Descripción:** Panel de visualización de métricas, KPIs y estado general del sistema.

**Componentes:**
- Indicadores clave de rendimiento (KPIs)
- Gráficos y estadísticas
- Mapa de situación actual
- Resumen de incidentes
- Estado de unidades y personal
- Filtros temporales
- Exportación de informes
